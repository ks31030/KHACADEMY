package란?
어떠한 폴더 위치를 작성하는 공간.
내가 만들고 싶은 파일 경로(폴더 위치)를 설정할 수 있다.

자바 명명 규칙
자바의 명칭 규칙은 클래스나 패키지, 변수, 상수, 메소드 등
자바 프로그래밍을 할 때 이름을 따라 할 규칙

카멜 케이스 (javaName)
이름에 여러 단어를 혼합해서 사용한다면 첫번째 단어는 소문자, 두번째 단어부터는 대문자로 시작한다.
이름을 지을 때 숫자를 먼저 입력해도 안 된다.
ex)
firstName
lastName
phoneNumber

스네이크 케이스 (java_name) 
모든 문자의 첫자는 소문자로 표기하고 문자들 사이에는 _(언더바)로 연결한다.
ex)
first_name
last_name
phone_number

파스칼 케이스 (JavaName)
모든 문자의 첫자를 대문자로 표기. 중간에 대문자를 넣어 표기
ex)
FirstName
LastName
PhoneNumber

케밥 케이스 (java-name)
모든 문자의 첫자를 소문자로 표기. 문자들 사이에는 -(하이픈) 연결.
ex)
first-name
last-name
phone-number
